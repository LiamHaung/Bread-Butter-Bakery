# 礁溪奶油麵包｜品牌形象與體驗升級提案

法式浪漫 × 宜蘭在地的單頁提案網站。

## 視覺規範

- **底色**：米白 (`#F8F5F0`)
- **文字**：深灰綠 (`#2D3832` / `#46594E`)
- **重點**：焦糖色 (`#B8956B` / `#C4A35A`)

## 使用方式

1. 直接用瀏覽器開啟 `index.html`
2. 或使用本地伺服器（可選）：
   ```bash
   npx serve .
   # 或
   python3 -m http.server 8000
   ```

## 自訂背景圖

首屏主視覺目前使用 Unsplash 示意圖。若要換成自己的圖片：

- 將圖片放在專案資料夾（例如 `hero.jpg`）
- 在 `styles.css` 中將 `.hero-bg` 的 `url(...)` 改為 `url("hero.jpg")`

## 檔案說明

- `index.html` — 提案頁面結構與文案
- `styles.css` — 版型與視覺樣式
- `script.js` — 輕量互動（可選）
