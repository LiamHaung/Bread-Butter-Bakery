# 圖片命名與路徑

## 視覺識別 Logo 輪播（三張）

01 視覺識別系統區塊的輪播圖：

| 張數 | 檔名 | 說明 |
|------|------|------|
| 第 1 張 | `logo-1.jpg` | 必填 |
| 第 2 張 | `logo-2.jpg` | 必填 |
| 第 3 張 | `logo.jpg`（或 `logo-3.jpg`） | 可另檔 |

- 第 1、2 張使用 `logo-1.jpg`、`logo-2.jpg`，第 3 張使用 `logo.jpg`；若要第三張不同圖可改為 `logo-3.jpg`。

---

## 伴手禮指引地圖輪播（五張）

03 伴手禮指引地圖 + 地貼設計區塊的輪播圖，請將圖檔放在此資料夾：

| 張數 | 檔名 |
|------|------|
| 第 1–5 張 | `map1.jpg`、`map2.jpg`、`map3.jpg`、`map4.jpg`、`map5.jpg` |

---

## 禮盒互動小卡（三欄，hover 翻面）

請將小卡圖檔放在此資料夾，並使用以下檔名（副檔名可用 `.png` 或 `.jpg`）：

**正面（預設顯示）**

| 欄位 | 檔名 | 說明 |
|------|------|------|
| 左欄 | `card-flavor-note.png` | 味覺回憶卡 正面 |
| 中欄 | `card-craft.png` | 職人對話卡 正面 |
| 右欄 | `card-social-frame.png` | 透卡／拍照框 正面 |

**背面（hover 時翻面顯示）**

| 欄位 | 檔名 | 說明 |
|------|------|------|
| 左欄 | `card-flavor-note-back.png` | 味覺回憶卡 背面 |
| 中欄 | `card-craft-back.png` | 職人對話卡 背面 |
| 右欄 | `card-social-frame-back.png` | 透卡／拍照框 背面 |

- 路徑：`assets/images/`
- 建議比例：約 3:4（直式），例如 240×320px 或等比例放大。
